A6.2 Increase your word power: Extreme adjectives
quite and very are used to weaken and strengthen adjectives
• This soup is quite tasty but it needs a little more salt.
• This soup is very tasty, well done!
tasty is a base adjective: it can be weakened and strengthened.
delicious is an extreme adjective and it cannot be weakened; it can only be emphasised with
words like absolutely or completely
• This soup is absolutely delicious!
Extreme adjectives usually have a base adjective partner with a related meaning, for instance
tasty and delicious.
absolutely
________________________________________ delicious
quite tasty tasty very tasty
Some adjectives can be both base or extreme and can be modified with all of these words.
• My sister is very different from me.
• My sister is completely different from me.
